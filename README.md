# bioskopDoliBel
Zadatak je uradjen kao dio pripreme ispita iz Tehnika i metoda programiranja.
WPF aplikacija daje prikaz informacija o filmovima koji su na repertoaru u bioskopu. 
Informacije o filmovima su dobijene iz baze podataka ciji je .sql fajl datu folderu database i taj fajl je neophodno importovati u Php MyAdmin.
